## Quick Start

### 1. Go to [blink.new](https://blink.new)

### 2. Paste the prompt

Copy the entire contents of [`PROMPT.md`](PROMPT.md) and paste it into blink.new.

### 3. Set your area

Replace `[ENTER YOUR CITY / NEIGHBORHOOD]` with your target location:

```
Santa Monica, CA
```

### 4. Let it run

The AI will:
- Scan Google Maps for businesses without websites
- Filter for real, active businesses (5+ reviews, has phone number, independently owned)
- Build a premium website for each one — parallax hero, animations, responsive, real copy
- Output a lead dashboard with contact info and best call times
- Give you a cold call script with objection handling

<br/>

## What Gets Built

For each lead, the AI builds a production-ready website with:

| Section | What's In It |
|---------|-------------|
| **Hero** | Full-viewport parallax image, business name, tagline from reviews, CTA button |
| **About** | Original copy that sounds like a human copywriter — references real details |
| **Services/Menu** | Built from listing data and review mentions with realistic prices |
| **Gallery** | Stock images matched to the business type (marked for owner to replace) |
| **Reviews** | Real quotes pulled from Google reviews with names and ratings |
| **Location** | Address, hours, click-to-call phone, map embed placeholder |

Every site is designed to look like a **$5,000 agency build** — not a template, not a demo.

## The Cold Call Script

The prompt includes a complete cold call framework:

**Opening** → Get to the point in 10 seconds. Tell them you already built their site.

**If they look** → Offer hosting + management for $/month. No contracts.

**Objection handling** → Responses for "I don't need a website," "How much?", "I'll think about it," and "I already have someone."

**Follow-up sequence** → Day 0, Day 2, Day 5, Day 10 cadence with exact text messages.

<br/>

## Tips

- **Mix business types** — don't pull 10 restaurants. Variety = more closes.
- **Call during slow hours** — the dashboard includes best call times based on their listed hours.
- **Lead with the site** — "I already built you a website" is the only opener that works. Don't pitch before they see it.
- **Text the link** — always offer to text the site URL. Visual proof converts better than phone descriptions.
- **Track status** — update the dashboard after every call so you know where each lead stands.
